# Simple Text Diff Tool

一個簡單、離線優先的文字比對工具。
A simple, offline-first text comparison tool.

## Features (功能)
- **Offline First**: 完全離線執行，保障資料隱私。 (Works completely offline, ensuring data privacy.)
- **Split View**: 左右對照視窗。 (Side-by-side comparison.)
- **Drag & Drop**: 支援拖拉 `.txt`, `.log` 檔案直接讀取。 (Supports dragging and dropping text/log files.)
- **Instant Diff**: 快速比對並高亮差異。 (Fast difference highlighting.)
- **Dark Mode**: 預設深色模式，保護眼睛。 (Default dark mode.)

## Usage (使用方法)

### Quick Start (快速開始)
Simply open `offline_diff.html` in your browser. No installation or server required.
直接在瀏覽器中開啟 `offline_diff.html` 即可使用。無需安裝或架設伺服器。

### Development (開發)
If you want to modify the source code and rebuild:
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. (Note: The offline file is manually packaged from the source files when needed)


## Privacy Note
This tool runs entirely in your browser. No data is sent to any server.
此工具完全在瀏覽器中執行，任何資料都不會上傳至伺服器。
